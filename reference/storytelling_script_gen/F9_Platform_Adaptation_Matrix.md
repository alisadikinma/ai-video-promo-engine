# F9_Platform_Adaptation_Matrix.md
## Platform-Specific Video Optimization — Same Story, Different Container
> **Priority:** 🟠 HIGH — Platform-agnostic scripts underperform on every platform.
> **Core Principle:** StoryBrand = message architecture (WHAT to say). Platform rules = attention architecture (HOW to hold them while saying it).

---

## 1. Why Platform-Agnostic = Underperforming

Each platform has different algorithm behavior, audience expectations, and technical requirements. A 30-second script optimized for TikTok will underperform on LinkedIn by default. The content stays the same — the **container** adapts.

## 3. Platform-Specific Retention Checkpoints

> **Universal retention science applied per platform.**

| Checkpoint | Target Retention | What Must Happen | Platform Notes |
|---|---|---|---|
| **0–1.7s** | 90%+ | Pattern Interrupt — triple-layer (visual + verbal + audio) | TikTok: most critical. LinkedIn: slightly more forgiving. |
| **3s** | 80–85% | Hook landed — problem stated or curiosity activated | YouTube Shorts sweet spot. Facebook "hook rate" metric. |
| **7s** | 70–75% | Foreshadow complete — open loop active | If retention drops below 65% here, the foreshadow failed. |
| **15s** | 60–65% | First micro-payoff delivered | TikTok Dec 2024 update shifted engagement threshold here. |
| **30s** | 50–55% | Mid-hook or re-engagement beat | The "sag zone" — requires deliberate re-engagement. |
| **60s+** | 45–50% | Value delivery sustained, Peak approaching | Only relevant for Manifesto/Explainer formats. |

### 3.1. The "Sag Zone" Recovery Techniques (15–30s Mark)

| Technique | Execution | Platform Fit |
|---|---|---|
| **Second Hook** | Re-state the open loop or introduce a new one | All platforms |
| **Format Shift** | Switch from VO to text card, or from face to B-roll | TikTok, Reels |
| **Energy Spike** | Music shift, pacing change, unexpected visual | All platforms |
| **Micro-Testimony** | 3-second customer quote inserted | LinkedIn, Facebook |
| **Bold Text Card** | Full-screen stat or claim | Mute-heavy platforms (LinkedIn, Facebook) |

## 4. "Same Story, Different Container" Adaptation Workflow

### Step 1: Start with the Longest Format
Create the master script in the longest needed format (Explainer/Manifesto). This is the "source of truth."

### Step 2: Extract Modular Components
Tag each section of the master script:
- `[HOOK]` — isolate the opening 5 seconds
- `[FORESHADOW]` — isolate the open loop
- `[BODY-KEY-POINT-1/2/3]` — tag each major value delivery
- `[PEAK]` — isolate the climax moment
- `[CTA]` — isolate the call to action
- `[WON DAY]` — isolate the closing identity image

### Step 3: Reassemble per Platform Rules
| Target Platform | Assembly Logic |
|---|---|
| **TikTok** | Hook (1.7s) + Foreshadow (2s) + Body compressed (10–15s) + Peak (3s) + CTA (3s). Maximum 30s. |
| **YouTube Shorts** | Hook (3s) + Foreshadow (3s) + Body (15–20s) + Peak (3s) + CTA (3s). Loopable ending preferred. |
| **Instagram Reels** | Hook (2s) + Foreshadow (2s) + Body (10–15s) + Peak (3s) + CTA (3s). Aesthetic-first. |
| **LinkedIn** | Hook (3s) + Foreshadow (3s) + Body full (20–30s) + Peak (5s) + CTA (5s). Burned-in captions mandatory. |
| **Facebook** | Hook (3s) + Body compressed (15–20s) + Peak (5s) + CTA with Lead Form (5s). Burned-in captions mandatory. |

### Step 4: Verify Adaptation
- [ ] Pacing matches platform target (cut frequency)
- [ ] Captions match sound environment (burned-in where needed)
- [ ] CTA matches platform convention
- [ ] Duration within platform optimal range
- [ ] Retention checkpoints addressed at correct timestamps

## 5. Retention Editing Specifications

### 5.1. Cut Frequency

| Format | Cut Interval | Notes |
|---|---|---|
| Short-form (≤30s) | Every 2–4 seconds | Higher frequency = higher retention on TikTok/Reels |
| Medium-form (30–90s) | Every 3–6 seconds | Balance between energy and comprehension |
| Long-form (90s+) | Every 4–8 seconds | Allow more breathing room for complex content |

### 5.2. Visual Stimulus Change Rules

- **New visual element every 3–5 seconds minimum** — angle change, graphic, text, B-roll switch
- **B-roll pacing:** 1.2–1.5× speed for energy (standard B-roll feels slow on social)
- **Text on screen:** Maximum 6 words per text card. 2–3 seconds display time.
- **Jump cuts:** Acceptable in UGC and Teaser formats. Forbidden in Manifesto.

### 5.3. "Dead Space Elimination" Protocol

**Definition:** Any frame/second with no narrative function — it doesn't advance the story, deliver value, or create emotion.

**Detection Test:** If removing the moment doesn't break the story → it's dead space.

**Common Dead Space:**
- Long establishing shots without VO or text
- Redundant B-roll that repeats the same idea
- Repeated information already stated
- "Um/uh" gaps in UGC (edit out)
- Logo animations longer than 1 second

**Rule:** In short-form, zero tolerance. In long-form, maximum 2 seconds per instance.

### 5.4. Loopable Content Design (YouTube Shorts)

For maximum algorithmic push, design the ending to flow seamlessly into the beginning:

| Technique | Execution |
|---|---|
| **Audio Loop** | Final word/phrase connects to first word. Music beat aligns. |
| **Visual Loop** | Final frame's composition mirrors opening frame. |
| **Narrative Loop** | End on a question or tease that the beginning answers. |
| **Motion Loop** | Camera movement at end matches movement at start. |

**Target:** Average view duration above 100% (viewers watch more than once without realizing).
